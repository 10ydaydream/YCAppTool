### 01.基础概念介绍
#### 1.1 如何加载高清大图
- 遇到的问题描述
    - 通过压缩图片可以取得很好的效果，但有时候效果就不那么美好了，例如长图像清明上河图，像这类的长图，如果我们直接压缩展示的话，这张图完全看不清，很影响体验。
- 如何解决加载大图
    - 可以采用局部展示，然后滑动查看的方式去展示图片。利用BitmapRegionDecoder来局部展示图片的，展示的是一块矩形区域。
